# AutomateEmailSending

Full-Stack Developer outreach email automation tool.

## Setup

1. Install dependencies: `dotnet restore`
2. Copy `appsettings.example.json` to `appsettings.json`
3. Fill in your Gmail credentials and file paths
4. Place your resume PDF and emails.txt in the project root
5. Run: `dotnet run`

## Required Files

- `govind_cv.pdf` - Your resume (add locally)
- `emails.txt` - List of emails (add locally)

## Security

Never commit `appsettings.json` with real credentials!
